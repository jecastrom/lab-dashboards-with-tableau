To see which data elements we need to create any plot, on the top right hand side conner click on the `Show me` option and hover the mouse cursor over the type of plot we want. Then at the bottom we will see the elements we need.

== Create a cross table with Marital status and Gender.

Press and hold `Ctr` then select the `Gender`, `Marital Status` and `Customers(Count(Distinct))` fields, then click on `Text Tables.` To colour the table, select the `Customers(Count(Distinct))` field, drag it and drop it on the `Colour` section under `Marks`. Next to fill the cells with colour, set the `Mark` from `Automatic` to `Square`.

Then under `Mark` click on `Colour` => `Edit Colours`.

Finally from the `Palette` scrolling menu select `Temperature Diverging`.
